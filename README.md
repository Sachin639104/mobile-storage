# Mobile Store Management System

A full-stack mobile store management app built with React, Node.js and MongoDB.

## Features
- Login/Logout with OTP reset
- Inventory management
- Search by model/brand
- Sales with receipt (Discount, GST, Payment Mode)
- Sales History with Revenue tracker
- Return/Exchange management
- Staff login (role based access)
- Festival notifications
- Low stock alert

## Tech Stack
- Frontend: React, Tailwind CSS, GSAP
- Backend: Node.js, Express
- Database: MongoDB Atlas
- SMS: Twilio

## Setup
1. Clone the repo
2. Install dependencies - npm install
3. Add .env file with your credentials
4. Run backend - npm start
5. Run frontend - npm run dev
